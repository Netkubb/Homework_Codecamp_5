value ใน display property มีอะไรบ้าง
  1. inline
  2. block
  3. content
  4. flex
  5. grid
  6. inline-*
  7. run-in
grid และ flex แตกต่างกันอย่างไร
gridทำเป็นตารางที่มีrow columnตั้งค่ารูปแบบต่างๆได้ง่าย
flexจะเรียงแบบทั่วไปไม่ได้fixเป็นตาราง แต่จะมีความยืดหยุ่นมากกว่า เช่นตรงกลางเว้นว่างได้
justify and align แตกต่างกันอย่างไร
justifyจัดการpositionในแกนหลัก(แกนนอน)
alignจัดการpositionแกนรอง(แกนตั้ง)
